#Flö

Read at 300WPM and Faster! (Open-Source Version of Spritz)

![alt tag](http://i.imgur.com/4UUTTxR.png)


##Usage
*Currently CLI-only!*

Ensure all files are in the same directory and then run:

    php flo.php
    
The text will immediately begin cycling.

##Required
* PHP ≥ 4.01
* Fixed-Width Font
